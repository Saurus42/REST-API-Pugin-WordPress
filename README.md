# REST-API-MENU
Adding menus endpoints on WP REST API v2

## Routes
Get all menus:

    GET /menus/v2/menus
    POST /menus/v2/menus
    PUT /menus/v2/menus
    DELETE /menus/v2/menus

Get all sub-menu:

    GET /menus/v2/sub-menus
    POST /menus/v2/sub-menus
    PUT /menus/v2/sub-menus
    DELETE /menus/v2/sub-menus
